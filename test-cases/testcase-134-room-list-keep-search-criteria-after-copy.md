No.134 部屋一覧/Room list - Keep search criteria after copying
Đối tượng
部屋一覧/room list screen - コピー/copy

Khái quát
Hiện tại, đang có sự khác biệt về hành vi giữ điều kiện search khi thực hiện 'Copy' tại màn hình 部屋一覧/Room list.

Trường hợp search bằng 物件コード/Propety code: 
Sau khi thực hiện copy và refresh lại màn hình, property code đã nhập vẫn được giữ và kết quả tìm kiếm đó vẫn được hiển thị.
Trường hợp search bằng 物件名/Property name: 
Sau khi thực hiện copy và refresh lại màn hình, property name đã nhập sẽ bị clear và room list lại hiển thị tất cả các record.

Cần chỉnh sửa hành vi này sao cho
 Ngay cả khi tìm kiếm bằng Property name và thực hiện copy, thì search condition đã nhập vẫn được giữ và kết quả search vẫn hiển thị, tương tự như khi tìm kiếm bằng Property code.

Ngoài ra, hoạt động hiện tại của 物件名カナ/Property name Kana vẫn chưa được xác nhận, nhưng
nếu cần thiết, sẽ hỗ trợ tương tự như 物件名/property name

Goal
Trong màn hình 部屋一覧/Room list, ngay sau khi copy thì search condition của những item dưới sẽ được giữ lại, kết quả search theo điều kiện đó sẽ vẫn hiển thị. 

物件コード/property code
物件名/property name
物件名カナ/property name Kana
